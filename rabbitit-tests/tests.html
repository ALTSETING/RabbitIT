FAQ

      


      

        

          Чи треба реєстрація?
          
Ні. Тест проходиться одразу. Контакт — лише якщо хочеш план навчання.

        

        

          Як оцінюється результат?
          
За кількістю правильних відповідей + точками по темам (topics).

        

        

          Можна додати свої питання?
          
Так. Додаєш/редагуєш JSON у папці /questions.
